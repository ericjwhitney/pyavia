Packages
--------

.. toctree::
    :maxdepth: 1

    pyavia.aero: Aerodynamics, thermodynamics, fluid dynamics. <pyavia.aero>
    pyavia.containers: Useful, less common containers not available in the standard library. <pyavia.containers>
    pyavia.data: Data manipulation, interpolation, filtering, etc <pyavia.data>
    pyavia.fortran: Algorithms and containers mimicking Fortran-style behaviour. <pyavia.fortran>
    pyavia.geo: Aeronautic-specific geometry functions / objects. <pyavia.geo>
    pyavia.iter: Handy functions for dealing with various iterables. <pyavia.iter>
    pyavia.math: Useful mathematical functions, particularly where not available in NumPy. <pyavia.math>
    pyavia.prop: Propulsion analysis such as reciprocating engines, propellers and gas turbines. <pyavia.prop>
    pyavia.solve: Functions for solving systems of equations and minimisation. <pyavia.solve>
    pyavia.struct: Structures, stress analysis, fatigue and damage tolerance. <pyavia.struct>
    pyavia.types: Functions for changing data between different / unusual types. <pyavia.types>
    pyavia.units: Units-aware calculations. <pyavia.units>
    pyavia.util: Small, general purpose utility functions. <pyavia.util>
